# 📝 To-Do List DEV

Um projeto simples e funcional de lista de tarefas (To-Do List) desenvolvido com **HTML, CSS e JavaScript puro**, com persistência de dados no navegador.

---

## 🚀 Funcionalidades

* ✅ Adicionar tarefas
* ✏️ Editar tarefas
* ❌ Remover tarefas
* 📅 Definir data de conclusão
* ✔️ Marcar como concluída (checkbox)
* 💾 Salvamento automático com `localStorage`
* 🔄 Carregamento automático ao abrir a página
* 📆 Exibição da data atual

---

## 🖥️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 Estrutura do projeto

```
📁 projeto
 ├── index.html
 ├── style.css
 └── script.js
```

---

## ⚙️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/maycondouglasbm10/projetos-frontand.git
```

2. Acesse a pasta:

```bash
cd seu-repositorio
```

3. Abra o arquivo `index.html` no navegador.

---

## 💡 Como usar

1. Digite uma tarefa no campo de texto
2. Escolha uma data de conclusão
3. Clique em **Adicionar**
4. Use:

   * ✔️ Checkbox → marcar como concluída
   * ✏️ Editar → alterar tarefa
   * ❌ Remover → excluir tarefa
