# Calculadora JS

Uma calculadora simples e funcional desenvolvida com HTML, CSS e JavaScript. Este projeto demonstra operações matemáticas básicas em uma interface web intuitiva.

## Funcionalidades

- **Operações Básicas**: Soma (+), subtração (-), multiplicação (×) e divisão (÷)
- **Limpar**: Botão "C" para limpar o display
- **Igual**: Botão "=" para calcular o resultado da expressão
- **Interface Responsiva**: Design centrado e adaptável

## Como Usar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em qualquer navegador web moderno.
3. Clique nos botões para inserir números e operadores.
4. Pressione "=" para calcular o resultado.
5. Use "C" para limpar o display.

## Tecnologias Utilizadas

- **HTML**: Estrutura da página
- **CSS**: Estilização com Tailwind CSS
- **JavaScript**: Lógica da calculadora

## Estrutura do Projeto

- `index.html`: Arquivo principal da página web
- `style.css`: Estilos personalizados
- `script.js`: Lógica JavaScript da calculadora

## Pré-requisitos

Nenhum. Basta um navegador web para executar o projeto.
