## Descrição

Este repositório contém uma coleção de projetos frontend desenvolvidos para demonstrar habilidades em desenvolvimento web. Cada projeto é uma aplicação simples e independente, construída utilizando as tecnologias fundamentais da web: **HTML**, **CSS** e **JavaScript**.

### Objetivo

O objetivo principal deste repositório é servir como um portfólio de projetos frontend, permitindo a prática e o aprimoramento de conceitos básicos e avançados em desenvolvimento web. Além disso, os projetos podem ser utilizados como exemplos educacionais ou como base para novos desenvolvimentos.

## Projetos Incluídos

Aqui está uma lista dos projetos disponíveis neste repositório, com uma breve descrição de cada um:

### 1. Calculadora JS
- **Descrição**: Uma calculadora funcional implementada em JavaScript puro, capaz de realizar operações matemáticas básicas como adição, subtração, multiplicação e divisão.
- **Tecnologias**: HTML, CSS, JavaScript
- **Arquivos**: `index.html`, `script.js`, `style.css`, `README.md`
- **Como usar**: Abra o arquivo `index.html` em um navegador web para interagir com a calculadora.

### 2. Formulário
- **Descrição**: Um formulário web simples para coleta de dados do usuário, com validação básica e estilização responsiva.
- **Tecnologias**: HTML, CSS
- **Arquivos**: `index.html`, `style.css`, `readme.md`, pasta `img/`
- **Como usar**: Abra o arquivo `index.html` em um navegador web para visualizar e testar o formulário.

### 3. Gerador de Senhas
- **Descrição**: Uma ferramenta para gerar senhas seguras de forma aleatória, com opções de personalização como comprimento e tipos de caracteres.
- **Tecnologias**: HTML, CSS, JavaScript
- **Arquivos**: `index.html`, `script.js`, `readme.md`
- **Como usar**: Abra o arquivo `index.html` em um navegador web para gerar senhas.

### 4. Página de Login
- **Descrição**: Uma página de login completa, incluindo formulários para login, criação de conta e recuperação de senha esquecida.
- **Tecnologias**: HTML, CSS, JavaScript
- **Arquivos**: `index.html`, `creat-an-accont.html`, `forgot-password.html`, `script.js`, `style.css`, pasta `img/`
- **Como usar**: Abra o arquivo `index.html` em um navegador web para acessar a página de login.

### 5. Página de Login - GitHub
- **Descrição**: Uma página de login estilizada inspirada no design do GitHub, focada em interface e experiência do usuário.
- **Tecnologias**: HTML, CSS
- **Arquivos**: `index.html`, `style.css`, `readme.md`, pasta `img/`
- **Como usar**: Abra o arquivo `index.html` em um navegador web para visualizar a página.

## Pré-requisitos

Para executar qualquer um dos projetos, você só precisa de um navegador web moderno (como Chrome, Firefox, Safari ou Edge). Não há dependências externas ou servidores necessários, pois todos os projetos são estáticos.

## Como Executar os Projetos

1. Clone este repositório para o seu computador:
   ```
   git clone https://github.com/seu-usuario/projetos-frontand.git
   ```

2. Navegue até a pasta do projeto desejado.

3. Abra o arquivo `index.html` no seu navegador web.

   - Para projetos com JavaScript, certifique-se de que o JavaScript está habilitado no navegador.
