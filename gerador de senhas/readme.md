# 🔐 Gerador de Senhas

## 🧾 Descrição

O **Gerador de Senhas** é uma aplicação web simples que permite criar senhas seguras de forma personalizada.  
O usuário pode definir o tamanho da senha e escolher quais tipos de caracteres deseja incluir.

O projeto foi desenvolvido utilizando:
- HTML
- CSS com Tailwind CSS
- JavaScript puro

---

## 🚀 Funcionalidades

- 🔢 Definir tamanho da senha
- 🔠 Incluir letras maiúsculas
- 🔡 Incluir letras minúsculas (padrão)
- 🔢 Incluir números
- 🔣 Incluir símbolos
- 🔄 Gerar senha aleatória
- 📋 Copiar senha para área de transferência

---

## 🧠 Como funciona

O sistema segue os seguintes passos:

1. Captura as configurações do usuário (tamanho e opções)
2. Monta uma lista de caracteres disponíveis
3. Gera caracteres aleatórios com base nessa lista
4. Exibe a senha no campo de saída
5. Permite copiar a senha com um clique
